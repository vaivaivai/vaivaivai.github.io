Бизнес

Что такое Лид, Лидогенерация, Лидменеджмент, Конверсия, Себестоимость. Что такое CRM, виды CRM-систем.

Инструкция

CRM - это система управления отношениями с клиентами.

amoCRM - это самовнедряемая система для отдела продаж.

Основные функции amoCRM:

  • сбор и систематизация базы клиентов
  • задачи и напоминания менеджерам
  • автоматизация отдела продаж (цифровая воронка)
  • аналитика (по сделкам, по воронке продаж, по менеджерам)

Преимущества amoCRM:

  • Digital воронка
  • специализация только на универсальных задачах отделов продаж
  • автоматизация лидогенерации
  • универсальная воронка продаж
  • автоматизированное управление целями и задачами сотрудников
  • единый источник получения всей аналитики по продажам
  • гибкий API (возможность любых доработок)
  • возможность интеграции с популярными сервисами
  • простота и интуитивное понимание интерфейса
  • автоматизация отдела продаж (цифровая воронка)
  • возможность использования сразу после регистрации

Персональный аккаунт партнёра в amoCRM. Настройка субдомена. Пользователи и права. История посещений. API. Безопасность данных.

Инструкция

Статья по безопасности

1. Настройка субдомена

Настройки > Общие настройки > Поле “Адрес”

Изменить текущий субдомен на Ваше название (латиницей). Нажать “сохранить”.

2. Добавление пользователя

Статья по настройке пользователей

Настройки > Пользователи и права > Добавить пользователя

Добавляем нового пользователя. Для это пишем его Email, Имя и фамилию, придумываем пароль для входа в amoCRM сотруднику, настраиваем права по сделкам, контактам, компаниям.

Добавляем новую группу пользователей (“Добавить группу”), пишем название группы.

Перетаскиваем в нее менеджеров.

Удаляем ненужного менеджера, путем перетаскивания его в корзину. Переносим данные менеджера на другого пользователя.

Делаем сопоставление полей из файла с полями amoCRM. Создаем отсюда же новое поле “тип клиента”. Тип поля: список.

Нажимаем “далее”. Смотрим загруженные данные.

Подробнее о Digital воронке

Статья по настройке воронки продаж

Рекомендуемые этапы воронки для продажи amoCRM партнерами:

  • Холодные контакты
  • Есть интерес к продукту
  • Презентация проведена
  • Тестирует продукт
  • Отправлено КП (коммерческое предложение)
  • Принимает решение
  • Подписывает договор
  • Счет выставлен

Далее ­ переходит в статус “действующий клиент”, либо “закрыто” и “не реализовано”.
